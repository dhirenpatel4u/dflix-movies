# Movie TV Website

## Features

- Fetches movie list from Google Apps Script JSON
- Displays poster grid
- TV remote support
- Enter opens player
- Fullscreen iframe player
- Ready for GitHub
- Ready for Vercel

## Deploy

### GitHub

Create repository

```
movie-tv-app
```

Upload all files.

### Vercel

Import GitHub Repository

Framework Preset:

```
Other
```

Build Command

```
None
```

Output Directory

```
.
```

Deploy.
